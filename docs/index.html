Utility Page Directory

Quick links to the standalone tools in this folder. Each card explains what the page does and links directly to the HTML file.

ChatGPT Bookmarklet Generator

Creates a bookmarklet that injects a multi-line prompt template into the ChatGPT prompt box so you can save and reuse prompts quickly.

Open chatgpt_bookmarklet.html →

Clipboard Processor

Reads text from your clipboard, converts line breaks into <div>-based HTML, and writes both text and HTML back to the clipboard with status logging.

Open clipboard_format.html →

Clipboard Tool

Mobile-friendly clipboard console with buttons to read or write plain text and HTML, view capability support, and manage the editable buffer.

Open clipboard_manipulator.html →

JavaScript Minify / Format Tool

In-browser utility that prettifies or minifies JavaScript using Prettier and Terser so you can clean up or compress snippets without a build step.

Open /js-format →

Open Blank Helper

Triggers a popup to about:blank that shows a provided message (or the detected browser name) and then navigates back—useful for popup-behavior testing.

Open open_blank.html →

Suspend Viewer

Renders a link preview from URL parameters by applying title, canonical URL, favicons, and optional Open Graph image into the document.

Open suspend-manual.html →

HTML Editor

A lightweight HTML editor with rich-text controls, safe paste sanitization, live preview, and flexible HTML load/insert options for quick editing.

Open html-editor.html →
